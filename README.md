# Aminul Islam — Portfolio

Personal portfolio for **Aminul Islam**, Co-Founder of Inventive Lab Inc.

Built with Next.js, TypeScript, and Tailwind CSS. Content sourced from GitHub profiles [`mr-aminul`](https://github.com/mr-aminul) and [`aminulnv`](https://github.com/aminulnv).

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

| Command       | Description        |
| ------------- | ------------------ |
| `npm run dev` | Development server |
| `npm run build` | Production build |
| `npm start`   | Serve production   |

## Edit content

Update copy, projects, and links in `src/data/site.ts`.

## Resume

Open [/resume](http://localhost:3000/resume) to view the on-site CV.

## Refresh project screenshots

```bash
npm run capture:screenshots
```

Screenshots are saved to `public/projects/`.
